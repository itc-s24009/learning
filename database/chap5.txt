SELECT 列名,
　　　 列名 + 100,
       "内容"

・SELECTでは、列名、計算式、固定値の三種類が指定できる
・列名はその列を指定、計算式は計算結果（計算に列名が使える）、固定値（すべてが内容で埋まる）


INSERT INTO テーブル名(列名)
VALUE (計算式)

UPDATE テーブル名
SET 列名 = 計算式


・UPDATE,INSERT INTOでも計算式が使える


特殊な演算子
xxx || yyy

・出力は「xxxyyy」のように、文字列を連結する


SELECT 
CASE 列名か式 WHEN 値１ THEN 値２
　　          WHEN 値３ THEN 値４
ELSE 値
END AS 出力名,
列名
FROM テーブル名
WHERE 条件式

・このCASEの部分は列名として扱う
・CASEの後の場所が、値１なら、値２に、値３なら値４に置き換える
・それ以外なら、ELSEの後ろの値に置き換える
・出力名を列名として出力する


SELECT 
CASE WHEN 条件１ THEN 値１
　　 WHEN 条件２ THEN 値２
ELSE 値
END AS 出力名
FROM テーブル名
WHERE 条件式


・CASEの後の場所が、条件１に当てはまるなら、値１に、条件２に当てはまるならなら値２に置き換える
・それ以外なら、ELSEの後ろの値に置き換える
・出力名を列名として出力する


SELECT 関数
FROM テーブル名

・関数の結果を列名と扱うことができる。

関数例

SELECT LENGTH(列名) AS 出力名
FROM テーブル名
WHERE LENGTH(列名) > 10

・LENGHTの中にある列名の文字数を出力する
・条件式にも関数が使える（LENGTH(列名) > 10）
・LENとも書ける


SELECT TRIM(列名) AS 出力名
FROM テーブル名

・左右の空白を取り除いて出力する
・LTRIMは左側のみ、RTRIMは右側のみの空白を取り除く


UPDATE テーブル名
SET 列名１ = REPLACE(列名２,入れ替え元の文字,入れ替え先の文字)


・列名１に、列名にの入れ替え元の文字を入れ替え先の文字と入れ替えた内容に更新する


SELECT *
FROM テーブル名
WHERE SUBSTRING(列名,数字１,数字２) LIKE '%X%'


・列名の、数字１の場所の文字から数字２の場所の文字までを出力する
（列名,1,5）の場合、「おはようございます」→「おはようご」
・条件式に使うことができる


SELECT CONCAT(列名１,列名２)
FROM テーブル名

・列名１と列名２を連結する。「||」と同じ動きをする
・列名以外にも文字を入れたりすることもできる


SELECT ROUND(列名,-数字) AS 出力名
FROM テーブル名

・列名の内容を、数字で指定された桁数で四捨五入する
・TRUNC関数だと、切り捨てになる


SELECT POWER(列名,数字) AS 出力名
FROM テーブル名


・列名の内容を数字で指定された回数べき乗する
（列名,3）→列名^3


日時の関数（列名のようにそのまま使っても良い）
CURRENT_TIMESTAMP　→　年、月、日、時、分、秒 
CURRENT_DATE　→　年、月、日
CURRENT_TIME　→　時、分、秒


CAST(変換する値 AS 変換する型)  →　変換後の値

・値の型を変更する関数
・数字型の入ってる列名を「数字 + '円'」にしたいとき、数字と文字なので合わせれない
　なので、CAST(数字型の入ってる列名 AS VARCHAR(20) + '円')でできる

・VARCHAR(10) →　0〜10文字分、文字型が入る型
・CHAR(10) →　10文字分、文字型が入る型（余った部分の後側はスペースで埋める）


COALESCE(列名１,列名２,列名3)

・左からチェックし、一番最初に出たNULLじゃない値を出力する

COALESCE(列名１,'XX')

・入っていればそのまま出力、NULLだったら'XX'で埋めて出力する
