SELECT 列名
FROM テーブル名
WHERE 列名 (SELECT 集合演算(列名) FROM テーブル名)

・SELECTの結果を別のSELECTで使いたい場合、（）で囲って使うことができる
・副問い合わせという

SELECTによる出力には三種類あり、それぞれの副問い合わせに種類がある

○単一行副問い合わせ
出力が一つの値である場合

・SELECTでの列を埋める単語
SELECT (副問い合わせの結果)
FROM テーブル名

・UPDATE SET での更新内容
UPDATE テーブル名
SET 列名 = (副問い合わせの結果)

・WHEREでの条件式での使用
SELECT 列名
FROM テーブル名
WHERE 列名 = （副問い合わせの結果）


○複数行副問い合わせ
出力が一列の場合

・WHERE INやANY,ALLを使った複数との比較
SELECT 列名
FROM テーブル名
WHERE 列名 IN (副問い合わせの結果)


○表副問い合わせ
出力が複数列、表の場合

・FROMの先のテーブルとして使える（ASを使い、テーブル名のようなものを作り、「表示名.列名」という表現をする）
SELECT 表示名.列名 
FROM (副問い合わせの結果) AS 表示名

・INSERTでの、複数同時登録
(特殊なので例を表示)
INSERT INTO テーブル名１(列名１、列名２)
SELECT テーブル名１の列名１に登録したいテーブル２の列名,テーブル名１の列名２に登録したいテーブル２の列名
FROM テーブル名２
WHERE 条件式
GROUP BY グループ分けの基準にしたいテーブル２の列名

のように、VALUEの部分をSELECT文にして登録できる


○相関問い合わせ
副問い合わせを使う式を式１、副問い合わせ自体を式２としたとき
式１で出たテーブルや列は、式２では使えない
しかし、副問い合わせの前にEXISTをつけると、使うことができる
これを相関問い合わせという。
（使用したテーブル名をつける（テーブル名.列名））

SELECT 列名
FROM テーブル名１
WHERE EXIST (SELECT * FROM テーブル名２WHERE テーブル名２.列名 = テーブル名１.列名)
