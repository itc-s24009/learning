１　文字Aテーブルから、「数字」の+10されたものを取得する

答え
select 数字 + 10 from 文字A;

普通に計算式を使う

２　文字Aテーブルから、「数字」が1なら10、それ以外は7に置き換えて取得する

答え
select case 数字 when 1 then 10 else 7 end as 数字 from 文字A;

caseからend as までを一つの列として扱い、caseの後ろにある列を
whenの後ろのものをthenの後ろに置き換えて、それ以外をelseの後ろに置き換える
end asで表示する列名を決める


３　文字Aテーブルから、「数字」が3以上なら10、それ以外は7に置き換えて取得する

答え
select case when 数字 >= 3 then 10 else 7 end as 数字 from 文字A;

caseからend as までを一つの列として扱い、caseの後ろから
whenの後ろの条件が当てまるものをthenの後ろに置き換えて、それ以外をelseの後ろに置き換える
end asで表示する列名を決める


４　文字Aテーブルから、「数字」の文字の長さを取得する

答え
select length(数字) from 文字A

関数lengthを使う


５　文字Aテーブルから、「ひらがな」を、左右の空白を取り除いて取得する

答え
select trim(ひらがな) from 文字A

関数trimを使う


６　文字Aテーブルから、「ひらがな」を、「あ」を「か」に置き換えて取得する

答え
select replace(ひらがな,'あ','か') from 文字A;

関数replaceを使う、カッコの最初に列名、置き換え前、置き換えあとと入れる


７　文字Aテーブルから、「ひらがな」を、１文字目から５文字目まで取得する

答え
select substring(ひらがな,1,5) from 文字A;

関数substringを使う。列名、最初の文字の場所、終わりの文字の場所、とかく


８　文字Aテーブルから、「ひらがな」と「カタカナ」を連結して取得する

答え
select concat(ひらがな,カタカナ) from 文字A;

関数concatを使う。


９　文字Aテーブルから、「数字」を、一桁目で四捨五入して取得する

答え
select round(数字,-1) from 文字A;

関数roundを使う。マイナスはつけるが、無視して見て、四捨五入する桁を書く


１０　文字Aテーブルから、「数字」を、二乗して取得する

答え
select power(数字,2) from 文字A;

関数powerを使う


１１　文字Aテーブルから、「ひらがな」と「数字」を、一番最初に出たNULLじゃない値を取得する

答え
select COALESCE(ひらがな,数字) from 文字A

関数coalesceを使う


１２　文字Aテーブルから、「ひらがな」を、NULLを「NULLだよ」に置き換えて出力する

答え
select COALESCE(ひらがな,'NULLだよ') from 文字A

関数coalesceは、列名、文字と書くと、NULLを置き換える機能がつく
