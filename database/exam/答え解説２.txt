１　都道府県テーブルから、すべての列を取得するコード

答え
select * from 都道府県;

fromでテーブルを選び、select * で、すべての行を取得する


２　都道府県テーブルから、「地域」と「都道府県名」を取得するコード

答え
select 地域,都道府県名 from 都道府県;

select部分で、取り出す列を選ぶ


３　都道府県テーブルから、「地域」が「東北」のものをすべて取得するコード

答え
select * from 都道府県 where 地域 = '東北';

fromの後ろにwhereで、条件式を使って絞る。文字列には''で囲む


４　都道府県テーブルに、(47,九州,沖縄,那覇,2273)を追加するコード

答え
insert into 都道府県 values (47,'九州','沖縄','那覇',2273);

insert intoでテーブルを選び、valuesで列を右詰めに追加する情報を()のなかに書く。


５　都道府県テーブルの、「県庁所在地」が「那覇」の行の「面積」を「2771」に更新するコード

答え
update 都道府県 set 面積 = 2271 where 県庁所在地 = '那覇';

updateでテーブルを選び、setで更新する列とその内容、whereで更新する列の条件を書く


６　都道府県テーブルの、「都道府県名」が「沖縄」の列を削除するコード

答え
delete from 都道府県 where 都道府県名 = '沖縄';

delete fromでテーブルを選び、whereで消す列の条件を指定する
