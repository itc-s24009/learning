１　都道府県テーブルから、「面積」が「5000」以上の列をすべて取得するコード

答え
select * from 都道府県 where 面積 >= 5000

２　都道府県テーブルから、「面積」が「3000」以下の列の「面積」と「都道府県名」を取得するコード

答え
select 面積,都道府県名 from 都道府県 where 面積 <= 3000;


３　都道府県テーブルから、「面積」が「3000」ではない列をすべて取得するコード

答え
select * from 都道府県 where 面積  <> 3000


４　都道府県テーブルから、「県庁所在地」に、「津」が含まれている列の「県庁所在地」を取得するコード

答え
select 県庁所在地 from 都道府県 where 県庁所在地 like '%津%';


５　都道府県テーブルから、「県庁所在地」が、「大」から始まり、二文字である列の「県庁所在地」を取得するコード

答え
select 県庁所在地 from 都道府県 where 県庁所在地 like '大_';


６　都道府県テーブルから、「県庁所在地」が、「_」から始まる列の「県庁所在地」を取得するコード

答え
SELECT 県庁所在地 FROM 都道府県 WHERE 県庁所在地 LIKE '$_%' ESCAPE '$';


７　都道府県テーブルから、「地域」が、「中国」もしくは「東北」である列のすべてを取得するコード
　　orを使うパターンと使わないパターンの２つ

答え
select * from 都道府県 where 地域 in ('中国','東北');
と
select * from 都道府県 where 地域 = '中国' or 地域 = '東北';


８　都道府県テーブルから、「面積」が、「2000」以下、もしくは、「面積」が「7000」以上である列のすべてを取得するコード

答え
select * from 都道府県 where 面積 <= 2000 or 面積 >= 7000;


９　都道府県テーブルから、「地域」が、「中国」かつ、「面積」が「7000」以上である列のすべてを取得するコード

答え
select * from 都道府県 where 地域 = '中国' and 面積 >= 7000;
