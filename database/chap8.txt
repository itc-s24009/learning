テーブル同士の関連→リレーションシップ


テーブルの結合

SELECT 列名
FROM テーブル名１
JOIN テーブル名２
ON テーブル１.比較する列名 = テーブル２.比較する列名 (例)

２つのテーブルを使って、出力する
ONに条件式を置き、その条件に合わせて結合する。NULLは使えない。

結合行の数がずれている場合
５月　１　　　　　食事　　　１
　　　　　　　　　お出かけ　１

のようになったとき、結合するときは、「５月」の行を複製して、
５月　１　食事　　
５月　１　お出かけ
のように結合される

なかった場合、そもそも結合、出力しない



結合条件の当てはまる相手がいなくても、テーブル１はすべて表示する。

SELECT 列名
FROM テーブル名１
LEFT JOIN テーブル名２
ON テーブル１.比較する列名 = テーブル２.比較する列名 (例)

結合条件の当てはまる相手がいなくても、テーブル２はすべて表示する。

SELECT 列名
FROM テーブル名１
RIGHT JOIN テーブル名２
ON テーブル１.比較する列名 = テーブル２.比較する列名 (例)

結合条件の当てはまる相手がいなくても、どちらのテーブルもすべて表示する。

SELECT 列名
FROM テーブル名１
RIGHT JOIN テーブル名２
ON テーブル１.比較する列名 = テーブル２.比較する列名 (例)

これら３つの結合では、条件に当てはまらず、結合する相手がない行が生まれます
そのような行では、隙間をすべてNULLで埋めて表示します


テーブルを結合する際、SELECTで列名を選択するときに、両方のテーブルに同じ名前の列があった場合
列名の名前の前に「.テーブル名」のようにすることで、引用元のテーブルを指定できる

SELECT テーブル名１.列名,テーブル名２.列名


３つ以上のテーブルの結合

SELECT 列名
FROM テーブル名１
JOIN テーブル名２
ON 条件式　　例：テーブル１.比較する列名 = テーブル２.比較する列名 
JOIN テーブル名３
ON 条件式


同じテーブルを結合する方法

SELECT 列名
FROM テーブル名１ AS 別名１
JOIN テーブル名１ AS 別名２
ON 条件式　　例：別名１.比較する列名 = 別名２.比較する列名 

と、ASを使って条件式を作る
