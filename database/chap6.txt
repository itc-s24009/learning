SELECT SUM(列名) AS 出力名
FROM テーブル名

・SUMで囲んだ列名のすべての合計を出力する

MAX → 最大値
MIN → 最小値
AVE → 平均値


SELECT COUNT(*) AS 出力名
FROM テーブル名
WHERE 条件式

・条件式に当てはまる行数を出力する
・COUNT(列名)でもほぼ同じ出力するが、選んだ列名にあるNULLの行は、行数に数えなくなる
・COUNT(DISTINCT 列名)という表記もできる


これら５つを集計関数という。SELECT文でしか使えない
→出力結果が一行なので、行の形がずれるから


SELECT 列名１ 集計関数(列名２) AS 出力名
FROM テーブル名
GROUP BY 列名３

・列名３を、同じ内容ごとにグループ分けして、
　そのグループごとの、列名１と、列名２の集計関数の結果を出力する


SELECT 列名１ 集計関数(列名２) AS 出力名
FROM テーブル名
GROUP BY 列名３
HAVING 集計関数(列名２) > 0

・グループ分けした際に、WHEREが使えない
（本来はFROMの下に置くが、GROUP BYまで行かないと集計が終わってないため
　集計が終わってない状態での条件指定の絞り込みは不可能だから）
・その代わりとなるHAVINGコマンド
・HAVINGの後ろにはいろんな条件式が置ける（集計関数も使える）
