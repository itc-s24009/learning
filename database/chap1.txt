SELECT 列名１,列名２,'XXX'
FROM テーブル名
WHERE 条件式


・指定したテーブルの中の列名を選んで表示する
・列名ではなく文字や数字を入れたら、その列はその単語で埋まる
・SELECT * ですべての列を表示
・複数の列を選んで表示したい場合、「,」で区切る
・条件式には、列名を使った条件を指定でき、SELECTされた中から条件に当てはまる列のみを表示する。
例：（列名１が50より大きいなら表示(列名１ > 50)）


INSERT INTO テーブル名
VALUES ('追加内容','追加内容','追加内容')

・選んだテーブル名に、データを追加する。
・列の順番の左詰めで追加される。
・追加内容には、''で囲まない数字情報、''で囲む文字列情報、'xxxx-xx-xx'の形の日付情報がある


UPDATE テーブル名
SET 列名１ = 変更したい内容
WHERE 列名２ = 変更したい列の条件


・WHEREの、列名２の条件が当てはまる列の、列名１を変更する。
・WHEREでの指定がない場合、すべての列を変更する。


DELETE FROM テーブル名
WHERE 列名 = 削除したい列の条件


・WHEREの条件が当てはまる列を削除する
・WHEREでの指定がない場合、すべての列を削除する
