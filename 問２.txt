class Human
Human→クラス

Human tanaka = new Human()
tanaka→インスタンスを格納、インスタンス化

int x
x→変数
また、変数などのデータのことを属性という

void jump()
jump →メソッド
また、メソッドなどの動きのことを操作という

クラスの最初にしていされているメンバ変数→フィールド

class cat{
    cat(String name){
        this.name = "tama";
    }
};

インスタンスの変数などの初期化、指定をする
cat(String name)
コンストラクタ、たくさん作れる

すでにあるクラスを引き継ぐ→継承
継承元クラス→スーパークラス、ひとつだけ
継承先クラス→サブクラス、たくさん作れる
スーパークラスに共通した性能をまとめること→汎化
サブクラスに独自の性能を入れること→特化

同じメソッドで違う結果→ポリモフィズム（多様性）

ファイル全体で使える変数→メンバ変数

メソッド内の変数を外部からいじれなくすることなど（変数にprivateなど）
外部からの操作を守ること→カプセル化
